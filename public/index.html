URL Shortener Microservice

Example creation usage:

https://ophid1an-urlshort.herokuapp.com/new/https://www.google.com
https://ophid1an-urlshort.herokuapp.com/new/http://freecodecamp.com/news

Example creation output:

{"original_url":"http://freecodecamp.com/news","short_url":"https://ophid1an-urlshort.herokuapp.com/2703"}

Usage:

https://ophid1an-urlshort.herokuapp.com/2703
